过滤器
======

Jinja2所有内置的过滤器都是可以使用的。 本文介绍的是定制的一些过滤器：

markdown
--------

用Markdown语法处理文本。

参数：

- ``codehilite`` : 是否高亮代码，默认为 ``True``

date
----

日期格式化，格式化选项与Python标准相同。

timestamp
---------

时间戳格式化，格式化选项与Python标准相同。

emphasis
--------

强调关键词，即给文本中出现的关键词加上 ``em`` 标签。

主要用于搜索时高亮显示搜索词。

参数：

- ``keyword`` : 要强调的关键词
